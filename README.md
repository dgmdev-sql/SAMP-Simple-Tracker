# Simple Tracker

A lightweight SA-MP Lua script to track player actions and job completions.

## Features

- Count job completions: detective, trucking, materials, garbage, selling cars, fishing and farming.
- Supports `/newb` or `/n` commands for helper tracking.
- View stats via `/trackstats`.
- Reset stats with `/trackreset`.

## Installation

1. Place `simple_tracker.lua` in your SA-MP `moonloader` folder.
2. Ensure `samp.events` is installed.
3. Run `/trackhelp` in-game for command list.

## Commands

- `/trackhelp` — Display this help.
- `/trackstats` — Show all tracked stats.
- `/trackreset` — Reset all stats to zero.
